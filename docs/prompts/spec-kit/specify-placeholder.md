# Create placeholder spec

Use for future work that should be tracked but not planned yet.

```text
/speckit.specify

SPECIFY_FEATURE_DIRECTORY={{SPEC_DIR}}

Create a placeholder spec for:

{{FEATURE_GOAL}}

This spec is not ready for planning or implementation.

Include:
- goal;
- why it is deferred;
- dependencies;
- open questions;
- non-goals;
- conditions required before promoting it to planning-ready.

Do not create or switch branches.
Do not implement code.
Do not run provider-backed commands.
```
